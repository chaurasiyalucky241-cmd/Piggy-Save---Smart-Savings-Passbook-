# 🐷 Piggy Save

## 💰 Smart Savings Passbook for Friends & Families

Piggy Save is a **modern digital savings application** inspired by online banking systems. It provides a beautiful and engaging user interface where users can manage savings, deposit money, withdraw funds, set savings goals, and maintain a digital passbook.

Designed with a cute mascot theme, glassmorphism effects, smooth animations, and responsive layouts, Piggy Save delivers a delightful user experience while showcasing modern frontend development.

---

# ✨ Features

- 🐷 Cute Piggy Mascot
- 🔐 Login & Registration System
- 💰 Digital Savings Passbook
- 📊 Interactive Dashboard
- 🎯 Savings Goal Tracker
- 💳 UPI Deposit Interface
- 💸 Withdrawal System
- 📜 Passbook History
- ⭐ Community Reviews & Ratings
- 🎁 Cashback Offer Card
- 🌙 Dark Mode
- 📱 Fully Responsive Design
- 🎨 Glassmorphism UI
- ⚡ Smooth Animations
- ❤️ Beautiful User Experience
