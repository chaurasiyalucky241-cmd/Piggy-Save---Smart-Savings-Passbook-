# 📖 About The Project

**Piggy Save – Smart Savings Passbook** is a creative frontend banking web application designed to make digital savings fun and engaging.

The project features a modern banking interface with adorable animated animal characters, soft gradients, interactive components, responsive layouts, and smooth user experience.

This project is completely built using **HTML**, **CSS**, and **JavaScript** without any external frontend framework.
